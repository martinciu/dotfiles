# Design: Add ~/.zshrc to dotfiles

**Date:** 2026-04-27
**Status:** Approved

## Goal

Track `~/.zshrc` and `~/.p10k.zsh` in the dotfiles repo, clean up the current config, and split machine-specific settings into an untracked `~/.zshrc.local`.

## File Structure

### Tracked in repo

| Repo path | Symlink target | Notes |
|---|---|---|
| `.zshrc` | `~/.zshrc` | Main shell config, cleaned up |
| `.p10k.zsh` | `~/.p10k.zsh` | Powerlevel10k prompt config |
| `.zshrc.local.template` | ‚Äî | Reference template, not symlinked |

### Untracked (per-machine)

| File | How created |
|---|---|
| `~/.zshrc.local` | Copied from `.zshrc.local.template` and filled in manually |
| `~/.secrets` | Existing ‚Äî already sourced by `.zshrc` |

## bootstrap.sh Changes

Add two `link` calls after the existing vim section:

```bash
link ".zshrc"     "$HOME/.zshrc"
link ".p10k.zsh"  "$HOME/.p10k.zsh"
```

Add a note in the `next steps` output reminding the user to create `~/.zshrc.local` from the template.

## .zshrc Contents (cleaned up)

Order and content of the tracked `.zshrc`:

1. **p10k instant prompt block** ‚Äî unchanged, stays at top
2. `export EDITOR=vim`
3. **oh-my-zsh** ‚Äî `$ZSH`, `ZSH_THEME=powerlevel10k/powerlevel10k`, `plugins=(git nvm brew rbenv node)`, `source $ZSH/oh-my-zsh.sh`
4. **Locale** ‚Äî `LANG=en_US.UTF-8`, `LC_ALL=en_US.UTF-8`
5. **PATH additions** ‚Äî cargo, asdf shims, `~/.local/bin` (all using `$HOME`, not hardcoded `/Users/martinciu/`)
6. **Project roots** ‚Äî `PROJECTS_HOME` only (`PROJECTS_ROOT` removed as redundant)
7. **load-nvmrc hook** ‚Äî `autoload -U add-zsh-hook` + hook function + `add-zsh-hook chpwd` + initial call
8. **p10k source** ‚Äî `[[ ! -f ~/.p10k.zsh ]] || source ~/.p10k.zsh`
9. **wt init** ‚Äî `if command -v wt ...; then eval "$(command wt config shell init zsh)"; fi`
10. **Source secrets** ‚Äî `[[ -f ~/.secrets ]] && source ~/.secrets`
11. **Source local** ‚Äî `[[ -f ~/.zshrc.local ]] && source ~/.zshrc.local` *(new)*

### Removals / fixes

| Item | Action | Reason |
|---|---|---|
| Manual NVM block (lines 34‚Äì36 of current) | Remove | `nvm` oh-my-zsh plugin already loads NVM; double-loading slows startup |
| All `/Users/martinciu/` paths | Replace with `$HOME` | Portability |
| `PROJECTS_ROOT` | Remove | Identical to `PROJECTS_HOME`; redundant |

## .zshrc.local Contents (machine-specific)

Items moved out of `.zshrc`:

- **conda/miniconda block** ‚Äî auto-generated by `conda init`, machine-installed
- **STM32 path** ‚Äî hardware-specific dev tool
- **ghcup env** ‚Äî Haskell toolchain, not universally present
- **Windsurf path** ‚Äî `~/.codeium/windsurf/bin`
- **opencode path** ‚Äî `~/.opencode/bin`
- **`CLAUDE_CODE_DISABLE_1M_CONTEXT=1`** ‚Äî personal preference, not portable

## .zshrc.local.template

Tracked in repo at `dotfiles/.zshrc.local.template`. Provides commented stubs for every machine-specific item so new machines know what to fill in. Not symlinked by bootstrap.

## What Is Not Changed

- `~/.secrets` ‚Äî stays untracked, sourced as-is
- oh-my-zsh installation ‚Äî assumed present, not managed by dotfiles
- `~/.p10k.zsh` ‚Äî copied from current machine as-is; no content changes
